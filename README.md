# Personal Portfolio Website  
### Built with React.js, Bootstrap, Axios & REST API Integration

This is the frontend of my personal portfolio website.  
It showcases my skills, projects, experience, certificates, and contact information in an interactive and responsive UI.

The website consumes APIs from my **Spring Boot backend**, making the content fully dynamic and easily manageable.

## Technologies Used

### **Frontend**
- **React.js**
- **React Router**
- **Axios** (API integration)
- **Bootstrap / Custom CSS**
- **React Icons**
- **SweetAlert2**---
